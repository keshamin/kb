# Schema Development

**Schema**: 
- Feature name
- Data type
- Required or optional
- Valency (for multivalue features)
- Domain or range
- Default value

## Schema Environment

Schema Environment means customizing schema basing on the current environment.  
Example: remove label column from serving environment.
